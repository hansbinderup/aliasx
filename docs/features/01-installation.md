# Installation & getting started

### Install

- Pre-built releases: GitHub Releases
- Arch (AUR): `yay -S aliasx`
- From source: `cargo install --git https://github.com/hansbinderup/aliasx/`

### Getting started

1. Create `.aliasx.yaml` in your repo root (optional)
2. Add some tasks
    - manually: see [Basics](02-basic.md)
    - automatically: use the [Config Generator](11-config-generator.md)
3. Run `aliasx` to explore and run tasks
4. Run `aliasx validate` to [validate](08-validation.md) the config(s)
