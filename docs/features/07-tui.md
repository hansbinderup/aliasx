# Interactive TUI

What the TUI does

The interactive TUI provides a fuzzy-search interface to find and run tasks and aliases. It is the default UI when running `aliasx` without flags.

Controls

- Type to filter
- `Arrow keys` : move selection
- `Enter` : run selected task
- `Esc` : exit
- `?` : toggle details in view
- `tab/⇧tab` : cycle through the scopes
